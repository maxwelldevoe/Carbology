# Team Members

* Max Devoe

# Installation

```

npm install react-search-input
npm install
bundle
rake db:create
rake db:migrate
rake db:seed

```

# Test Suite

```
rake
npm test

```


Then navigate to http://localhost:3000 in your browser.
